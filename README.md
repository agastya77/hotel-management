

<h1>Hotel Management System</h1>
<p><strong>Python + MySQL Project</strong></p>

<hr>

<h2>Project Description</h2>
<p>
The <strong>Hotel Management System</strong> is a console-based Python application
used to manage hotel operations such as room reservation, customer check-in,
check-out, and administrative record management.
</p>
<p>
This project uses <strong>MySQL</strong> as the backend database and demonstrates
Python–MySQL connectivity.
</p>

<h2>Technologies Used</h2>
<ul>
    <li>Python 3</li>
    <li>MySQL</li>
    <li>MySQL Connector for Python</li>
</ul>

<h2>Features</h2>

<h3>Customer</h3>
<ul>
    <li>Room reservation</li>
    <li>Suite selection</li>
    <li>Customer check-in</li>
    <li>Customer check-out</li>
    <li>Payment and reviews</li>
</ul>

<h3>Admin</h3>
<ul>
    <li>Admin login</li>
    <li>Add customer records</li>
    <li>Edit customer details</li>
    <li>Delete customer records</li>
    <li>Search customer records</li>
    <li>Display all customer records</li>
</ul>

<h2>Database Structure</h2>
<ul>
    <li>admin_login</li>
    <li>customer</li>
    <li>cust_personal</li>
    <li>check_in</li>
    <li>check_out</li>
</ul>

<h2>How to Run the Project</h2>

<h3>Step 1: Install MySQL Connector</h3>
<pre>
python -m pip install mysql-connector-python
</pre>

<h3>Step 2: Start MySQL Server</h3>
<p>
Ensure MySQL server is running and update credentials in the code if needed.
</p>

<h3>Step 3: Run the Project</h3>
<pre>
python hotel_project_final.py
</pre>
